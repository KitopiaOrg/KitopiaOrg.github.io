## 📖 简介

**Kitopia** 不仅仅是一个工具箱，它是一个**可高度自定义的生产力平台**。

与其他工具不同，Kitopia 引入了 **节点化编辑 (Node-based Visual Scripting)** —— 类似于 Unreal Engine 的蓝图系统。这意味着你不再受限于固定的插件功能，而是可以像搭积木一样，自由组合来自不同插件的原子功能，创造出完全符合你个人需求的自动化工作流。

### 核心亮点
- ✨ **开源免费**：代码透明，本地功能完全免费。
- 🧩 **高度插件化**：功能模块化，按需加载。
- 🔗 **节点化编程**：通过可视化连线定制你的专属功能 (Scenes)。
- 💻 **跨平台架构**：基于 Avalonia 开发 (目前优先支持 Windows)。

## ⚡ 主要功能

### 1. 全能搜索框 (Launcher)
> 默认快捷键: `Alt` + `Space`

不仅是启动器，更是你的系统控制台。

- **快速启动**：支持常规软件、UWP 应用、文件/文件夹。支持以管理员运行、打开文件位置等。
- **Everything 集成**：毫秒级搜索全盘文件。
- **剪贴板智能操作**：自动识别剪贴板内容（如路径），一键打开。
- **网页搜索**：快速打开网页或在搜索引擎中搜索。
- **命令行执行**：直接运行 Shell 命令。
- **实用小工具**：数学计算、剪贴板图片保存、微软翻译等。

### 2. 极致截图工具
> 默认快捷键: `Ctrl` + `Alt` + `Q`

支持 **HDR** 和 **长截图** 的专业级截图工具。

- **HDR 支持**：完美还原高动态范围画面，色彩不失真。
- **长截图**：支持窗口滚动截取。
- **丰富的标注工具**：矩形、圆形、箭头、画笔、文字、马赛克等。
- **OCR 文字识别**：截图即提取文字。
- **二维码识别**：自动识别图中二维码。
- **截图翻译**：直接翻译截图中的文本。
- **贴图 (Pin)**：将截图置顶显示在屏幕上。

### 3. 鼠标手势
> 默认快捷键: `长按鼠标侧键2`

通过简单的鼠标动作触发复杂操作。

![Mouse Gestures](./assets/README/5515238963800.png)

### 4. 节点化场景 (Scenes)
> **Kitopia 的灵魂功能**

编写你自己的“情景”！无需编写代码，通过拖拽和连接节点，调用软件自身或插件提供的功能。

- **可视化编辑**：直观的逻辑编排。
- **无限可能**：组合不同插件的能力，构建自动化任务。

![Node Editor](./assets/README/Kitopia1766548006092.png)
